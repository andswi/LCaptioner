I want to create a Pinokio application to caption images using local API to LM Studio. The application must support drag & drop of a multiple images dataset and an call to action button "Caption". For each of the images the application will call a local instance of LLM through an API (LM Studio) and send the image with instruction "Caption this image". When the caption is received it will be saved to a txt file in the same directory as image, with same name as its corresponding image file. Captioning already existing txt files will overwrite the old caption with new one.
